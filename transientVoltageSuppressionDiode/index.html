瞬态电压抑制二极管

瞬态电压抑制二极管，简称TVS(Transient Voltage Suppression Diodes)，用作防止电压突变导致电路元件损毁。主要特性为电压低时，自身电阻无限大，达到开路的效果；电压高时，自身电阻无限小，达到短路的效果，继而短接电源，保护电路。TVS 分为两类，单项与双向

主要参数

  • : Working peak reverse voltage 最大反向工作电压
    • 电压大于时，TVS是普通的二极管
    • 为了电路的正常运作，要保证最大反向工作电压高于工作电压
  • : Breakdown voltage 击穿电压
    • 当电压大于 时，TVS 会被击穿变为阻抗极低的导线
  • : Clamping voltage 钳位电压
    • TVS 会将 高于的尖端电压按低到钳位电压
  • : Reverse current 漏电流
    • 最大反向工作电压下的流过的最大电流
  • : Reverse breakdown current 反向击穿电流
  • : Peak pulse current 峰值脉冲电流
    • 当TVS 处在钳位电压时，可流过的最大电流
